About Lab 4

Flask Authentication and File Management System

This application was developed as part of the INFO3180: Web Design and Programming II course. It demonstrates the implementation of secure user authentication using Flask-Login and persistent data storage with PostgreSQL.

Key Features:

  • Secure User Login/Logout functionality.
  • Password hashing using Werkzeug security helpers.
  • Database migrations handled via Flask-Migrate.
  • Image upload system with secure filename handling.
  • Dynamic image gallery displayed using CSS Grid.
